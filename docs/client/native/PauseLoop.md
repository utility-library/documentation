# PauseLoop
Pause a loop, the loop stop to work but can be resumed later

| Argument              | Data Type                            | Needed                    | Default         | Description
| ----------------------| ------------------------------------ | ------------------------- |-----------------|-------------
| `loopid`                | number | :material-checkbox-blank-circle: | `-` | The loop id

!!! success ""
    Dont need to be called every frame
---
??? example
    ```
    PauseLoop(loopId)
    ```
