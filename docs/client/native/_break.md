# _break
Break the current loop

| Argument              | Data Type                            | Nedeed                    | Default         | Description
| ----------------------| ------------------------------------ | ------------------------- |-----------------|-------------
| `LoopId`                | number | :material-checkbox-blank-circle: | `-` | The loop id

---
???+ warning
    It can be used only into `CreateLoop` function
??? example
    ```
    _break(loopId)
    ```
