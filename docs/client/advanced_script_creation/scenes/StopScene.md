# StopScene
Stop an existing scene

| Argument              | Data Type | Needed                                   | Default | Description                                                                       |
|-----------------------|-----------|------------------------------------------|---------|-----------------------------------------------------------------------------------|
| `scene`               | string    | :material-checkbox-blank-circle:         | `-`     | The scene netid                                                                   |

!!! success ""
    Dont need to be called every frame
---
??? example
    ```
    StopScene(scene)
    ```
