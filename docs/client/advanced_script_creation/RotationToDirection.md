# RotationToDirection
Obtains the direction from a rotation

| Argument              | Data Type                            | Needed                    | Default         | Description
| ----------------------| ------------------------------------ | ------------------------- |-----------------|-------------
| `Rotation`                | vector3 | :material-checkbox-blank-circle: | `-` | The rotation

!!! success ""
    Dont need to be called every frame
---
??? example
    ```
    RotationToDirection(rot)
    ```
