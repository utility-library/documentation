    Task

    It will create a thread in the loop with a default time, can be stopped by the StopLoopThread using the id returned.

    Available Task

    Name Velocity
    TaskBack 5000 ms
    TaskSlow 1000 ms
    TaskFast 500 ms
    TaskExtraFast 5 ms
    Argument Data Type Needed Default Description
    LoopId number - The loop id
    Id string/number - The id to identify the thread
    Function function - The code executed in the loop thread
    Returns
    Data Type Description
    number The handle of the LoopThread

    Dont need to be called every frame

    Warning

    It can be used only into CreateLoop function

    Example
    CreateLoop(function(loopId)
    TaskBack(loopId, "main", function()
        print("Hello")
    end)
end, 250)
